# Astroid-miner

Open notebook and conceptual archive for asteroid mining, self-replicating swarms (Leviathan / Lazarus Forge), ISRU architectures, mass/energy budgets, zero-g fabrication, propulsion economy, and long-term planet-building ideas (Ceres seed, high-pressure cores, etc.).

**Current Status** (Jan 2026): Early conceptual phase. Focus on bootstrapping from near-Earth asteroids → main belt (Ceres hub) → self-sustaining replication. All documents are work-in-progress; contributions welcome via PRs or issues.
